# README screenshots

The main README references four images in this folder. Capture them from the
running app (`bun run tauri dev` or an installed build) for best fidelity.

Suggested setup: maximize the window, pick the **Dark** theme, and open
[`SHOWCASE.md`](./SHOWCASE.md) (it exercises every renderer feature).

| File | What to capture |
|------|-----------------|
| `split.png`    | **Split view** of `SHOWCASE.md` — source on the left, live preview on the right. |
| `palette.png`  | The **command palette** open (`Ctrl+Shift+P`) over the editor. |
| `math.png`     | The preview scrolled to the **Math + Mermaid** sections. |
| `settings.png` | The **Settings** dialog (e.g. the Appearance or Editor tab). |

Save them here as PNG with those exact names. Keep them reasonably sized
(~1280px wide is plenty); GitHub scales them in the table.
